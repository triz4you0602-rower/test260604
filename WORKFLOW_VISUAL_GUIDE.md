## 🎓 배운 점 및 핵심 기술

```
┌────────────────────────────────────────────────────────┐
│           프로젝트 핵심 기술 및 패턴                   │
└────────────────────────────────────────────────────────┘

1️⃣ 데이터 소스 다중화 (Multi-Source Pattern)
   - Supabase 우선 (실시간, 클라우드)
   - CSV 폴백 (안정성, 오프라인)
   - 자동 전환 메커니즘

2️⃣ 클라우드 DB 연동 (Cloud Integration)
   - RESTful API를 통한 Supabase 연결
   - 권한 관리 (Role-Based Access Control)
   - 실시간 데이터 쿼리

3️⃣ 배포 자동화 (CI/CD)
   - GitHub → Streamlit Cloud 자동 배포
   - Secrets 관리 (로컬 vs 클라우드)
   - 환경별 설정 분리

4️⃣ 예외 처리 및 복원력 (Resilience)
   - Try-Except 패턴
   - 자동 폴백 메커니즘
   - 사용자에게 명확한 피드백

5️⃣ 데이터 검증 (Validation)
   - 두 데이터 소스 동기화 확인
   - 실시간 동기화 테스트
   - 메트릭 정확성 검증
```
